# A股专业选股工具

本技能使用东方财富免费API获取A股数据，进行专业筛选。

## 筛选条件

1. **成交量趋势**：近30天成交量温和放大（可选）
2. **市盈率**：低于行业平均15%（可选）
3. **排除标的**：
   - ST股票
   - 退市风险标的
4. **其他条件**（可选）：
   - 主板/创业板/科创板
   - 上市时间
   - 市值范围

## 使用方法

### 1. 查看帮助
```
A股选股帮助
```

### 2. 执行筛选
```
A股筛选 [条件]
```

示例：
- `A股筛选 成交量放大` - 筛选近30天成交量温和放大的股票
- `A股筛选 低市盈率` - 筛选市盈率低于行业平均的股票
- `A股筛选 完整条件` - 使用所有筛选条件

## 技术实现

使用Python脚本调用东方财富API获取数据：
- 股票列表：`http://28.push2.eastmoney.com/api/qt/clist/get`
- 行业市盈率：`http://push2.eastmoney.com/api/qt/ulist.np/get`
- 个股数据：`http://push2.eastmoney.com/api/qt/stock/get`

## 输出格式

筛选结果将包含：
- 股票代码
- 股票名称
- 当前市盈率
- 所属行业
- 行业平均市盈率
- 30天成交量变化
- 最新价格
- 涨跌幅
